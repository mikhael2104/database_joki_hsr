# database_joki_hsr
Merupakan Website dashboard sederhana yang berguna untuk membuat toko jasa joki game honkai star rail. Dibuat dengan MongoDB, NoSQL, serta penggunaan Express JS


Apabila ingin menjalankan bisa masuk ke folder backend terlebih dahulu
ketikkan pada terminal <cd backend>

Kemudian ketikkan <node server.js> untuk menjalankan koneksi servernya

Setelah itu bisa menggunakan live server, bisa melalui installasi ataupun ekstension
1. Apabila installasi lewat NPM ketikkan <npm install live-server>
2. Apabila installasi lewat ekstension, bisa langung cari "live server"

Untuk yang melakukan installasi di NPM (Terminal), ketikkan live-server frontend
Maka website akan langsung berjalan.
